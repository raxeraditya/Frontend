# 💫 About Me:
## Overview<br>This repository contains the frontend client for my personal portfolio. It acts as a digital hub to demonstrate my full-stack web development capabilities, user interface designs, and featured software projects. <br><br>### Key Features<br>* ⚡ **Performance & SEO:** Built with Next.js App Router for server-side rendering, static site generation, and optimized metadata previews.<br>* 🎨 **Responsive UI:** Tailored with Tailwind CSS for mobile-first, adaptive displays across all viewports.<br>* 🌓 **Dark / Light Theme:** Custom theme switcher with smooth transitions.<br>* 📁 **Project Showcase:** Interactive project gallery with deep dives into architecture and dynamic routing.<br>* 📬 **Contact System:** Functional contact form integrated with server actions / API endpoints.
